 # 📦 Fullstack Inventory And Order Management System

This project is a **Fullstack Inventory and Order Management System** built with **React (frontend)**, **Node.js + Express (backend)**, and **MySQL (database)**.  
It demonstrates the core functionality of an inventory and order tracking platform, including authentication, role‑based authorization, product management, order tracking, and an admin dashboard.  
⚠️ Payment integration is **not included**, but the system showcases the essential features of a shopping and inventory management application.

---

## 🔑 Key Features
- **Authentication & Authorization**: Secure login with JWT, role‑based access (Admin vs Customer).
- **Product Management**: Admin can add, update, and delete products, including stock tracking.
- **Order Management**: Admin can view the orders 
- **User Profile**: Customers can view their order history and manage their account.
- **Admin Dashboard**: Displays total revenue, products sold, and sales statistics.
- **Responsive UI**: Built with React and TailwindCSS.

---

## ⚙️ Tech Stack
- **Frontend**: React, Axios, TailwindCSS, React Router
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Authentication**: JWT (JSON Web Token)

# 📦 Fullstack Stok ve Sipariş Yönetim Sistemi

Bu proje; **React (ön yüz)**, **Node.js + Express (arka yüz)** ve **MySQL (veri tabanı)** kullanılarak geliştirilmiş **Fullstack bir Stok ve Sipariş Yönetim Sistemidir**.  
Bir stok ve sipariş takip platformunun; kimlik doğrulama (authentication), rol tabanlı yetkilendirme (role‑based authorization), ürün yönetimi, sipariş takibi ve admin yönetim paneli gibi temel işlevlerini uygulamalı olarak gösterir.  
⚠️ Ödeme entegrasyonu **dahil değildir**, ancak sistem bir alışveriş ve stok yönetim uygulamasının en temel ve gerekli özelliklerini barındırmaktadır.

---

## 🔑 Temel Özellikler
- **Kimlik Doğrulama ve Yetkilendirme**: JWT ile güvenli giriş ve rol tabanlı erişim kontrolü (Admin ve Müşteri ayrımı).
- **Ürün Yönetimi**: Admin panelinden stok takibi dahil olmak üzere ürün ekleme, güncelleme ve silme işlemleri.
- **Sipariş Yönetimi**: Admin, siparişleri görüntüleyebilir.
- **Kullanıcı Profili**: Müşterilerin kendi sipariş geçmişlerini görüntüleyebilmesi ve hesaplarını yönetebilmesi.
- **Admin Yönetim Paneli**: Toplam gelir, satılan ürünler ve satış istatistiklerinin gösterimi.
- **Duyarlı (Responsive) Arayüz**: React ve TailwindCSS ile tüm cihazlara uyumlu tasarım.

---

## ⚙️ Kullanılan Teknolojiler (Tech Stack)
- **Ön Yüz (Frontend)**: React, Axios, TailwindCSS, React Router
- **Arka Yüz (Backend)**: Node.js, Express.js
- **Veri Tabanı (Database)**: MySQL
- **Kimlik Doğrulama (Authentication)**: JWT (JSON Web Token)
